# 一个让产品经理提前下班的 Claude Code Skill（自用版）

- 你是否还在深夜对着空白文档发呆，把"背景"两个字打了删、删了打？
  
- 你是否还在被开发追问"这个边界条件怎么处理"时心虚地说"这个我下来看看"？

- 你是否还在用"记个todo吧~"糊弄每一个你没想清楚的功能点？

现在，把需求丢给AI，它会帮你生成一份连你自己都觉得"卧槽这真是我写的？"的PRD
结构完整、逻辑清晰、边界条件穷举到开发都挑不出毛病——当然，需求变更的锅它不背
**（在梦里有）** 

### 支持两种模式：

- 脑子清醒时：一口气描述完，直接出稿
- 脑子浆糊时：AI引导你一步步把需求捋清楚

  **P0功能：帮你写PRD**

  **P3功能：帮你活过评审会（P3暂不支持）**
